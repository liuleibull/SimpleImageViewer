# SimpleImageViewer
Simple Xamarin.Android ImageViewer.

### Features
- sliding gallery;
- pinch-zoom;
- caching;
- infinite scroll;
- huge images.

### Libraries
- [FFImageLoading](https://github.com/luberda-molinet/FFImageLoading)
- [SubsamplingScaleImageView](https://github.com/Softeq/SubsamplingScaleImageView-xamarin-android)
